### Backwards-incompatible changes

- Documenteer now targets Sphinx 8 and 9. The core requirement is raised to `Sphinx>=8,<10`, so Sphinx 7 is no longer supported. Continuous integration tests against Sphinx 8 and 9, and a scheduled run additionally exercises Sphinx's development branch.

- Dependency requirements raised for the `guide` and `technote` extras:

  - `pydata-sphinx-theme>=0.19,<0.20` (was `>=0.16.1,<0.17.0`).
  - `sphinxcontrib-bibtex>=2.7.0`, which clears the old `<8.2` Sphinx gate.
  - `sphinx-design>=0.7.0`.
  - `myst-parser>=4.0.0` (the first release supporting Sphinx 8) and `myst-nb>=1.1.0`.
  - `technote>=0.9.1`.
